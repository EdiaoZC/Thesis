手机端登陆
>    
     向服务器 地址：/user/login 发送 **POST** 请求
     参数为： 
        username = *** password =***，
     登陆成功后会返回一个token,token值为uuid
>
向服务器请求运行所需参数
>
    登陆成功后，再次向服务器发送设备请求，
    参数为:
        deviceId:******
        username:******
        token:**********
    向服务器得请求后，会立即在 web 端产生响应，以便及时处理
    通过修改设备的运行参数，提交后向手机端发出回应。
>
